yaml represents config data. Similar to JSON on XML.

key value pairs in yaml

```yaml
Fruit: Apple
Vegetable: Carrot
Meat: Chicken
```

array:

```yaml
Fruits:
  - Orange
  - Apple
  - Banana
```

Dictionary/Map

```yaml
Banana:
  Calories: 105
  Fat: 0.4 g
  Carbs: 27 g
```

Spaces are important in yaml.Items on same level should have same number of spaces

```yaml
Fruits:
  - Banana:
      Color: Yellow
      Calories: 105
  - Grape:
      Color: Purple
      Calories: 50
```
